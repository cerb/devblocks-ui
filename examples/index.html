    Dynamic (Ajax) panels

    Tabs whose <a href="/path"> loads content via fetch() on first click. A spinner shows while loading; content is cached after the first load. Call tabs.refresh(index) to force a reload. The component auto-manages the panel <div> — you don't define it.

    Spinner

    Loading indicator

    CSS-animated spinner used internally by Tabs while dynamic panels load. Can also be used standalone — new Spinner() returns an object with an .el span you insert wherever you need it. Call spinner.destroy() to remove it.

    Toggle

    Binary switch

    Enhances an input[type=checkbox] with an animated binary switch. The native input is visually hidden but stays accessible — screen readers interact with it directly, and it still submits with its form. Click the track or the label text, or tab to it and press Space.
