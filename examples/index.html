    Basic dialog

    A floating window that wraps any existing element as its content area. Drag by the titlebar to reposition; grab any edge or corner to resize. The minimize button collapses the content and keeps the titlebar visible; click it again to restore. Escape closes the topmost open dialog.

    SelectMenu

    Basic

    Wraps a <select> element with a styled trigger and floating panel. The underlying <select> is hidden but stays in the DOM so its value submits with any form. Click the trigger or press Space/Enter/Arrow to open; click an item or press Enter to select; Escape closes.

    No selection yet.

    Large list with type-to-filter

    Open the menu and type to filter — for example, type Los or Ang to narrow a full timezone list down to matching entries. The filter text appears above the list; Backspace removes characters. Selecting an item (or pressing Escape) clears the filter. The currently selected value is unchanged until you explicitly pick a new one.

    Custom rendering (onRender)

    The onRender callback receives the rendered <li> and the source <option> before the item is inserted into the DOM, letting you prepend icons, badges, or any additional markup. The default label span is already present; add siblings around it.

    Accordion

    Basic accordion

    Enhances a <div> with alternating <h3> / <div> children into an accordion. Clicking a header expands its panel and collapses the previously open one. Arrow keys move focus between headers; Enter/Space toggles via the native button.

    Getting started

    Install via npm or drop the dist script tag into your page. No runtime dependencies.

    Keyboard navigation

    ArrowDown / ArrowUp move focus between headers. Home / End jump to the first or last. Enter and Space toggle the focused section (native button behaviour).

    Dark mode

    Set data-dui-theme="dark" on any ancestor to switch the palette. All colours are CSS custom properties.
